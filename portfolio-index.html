Universal Navigation System

Skills

  • C++
  • Object Oriented Programming
  • Graph Theory
  • Dijkstra Theorem
  • Depth/Breadth First Search
  • Server-Client communication

The project involved building a navigation system/path-finder program, in C++, which makes use of server-client connection. The Client would receive coordinates of the start and end points from a GUI based map of Edmonton. After establishing a connection with the server, it would send these coordinates to it. The server would make use of the above-mentioned algorithms to find the shortest path between these two points. If the path exists, it would send the coordinates to the client once at a time and then the client would write them to a pipe to be used by the map. The path would then get displayed on the map and the user could select another pair of points after that. If the path did not exist, the map would get 2 dots, with no path between them.

Object Oriented Design based Data Classifier

Skills

  • Python
  • Object Oriented Programming/li>
  • Pre-processing
  • k-cross validation
  • Accuracy
  • Precision
  • Recall

This project involved building an object-oriented weather data classifier in python. The data would be pre-processed by removing punctuation, helping words and numbers (unless specified otherwise). Next it calculated the frequency of the words. After that we would train the classifier based on the top n words. This would become our training set. Finally, we would perform k-fold cross validation on the same data set to check for precision, accuracy and Recall of the model.
